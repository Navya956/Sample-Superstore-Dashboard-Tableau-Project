# Sample-Superstore-Dashboard-Tableau-Project
This project focuses on creating an interactive sales dashboard using the Sample Superstore dataset in Tableau.

## Key Highlights
Analyzed sales, profit, and regional performance.
Created interactive charts, KPIs, and filters.
Identified category-wise business trends and insights.
Improved data visualization and dashboard design skills.
Used filters, slicers, and interactive features to improve dashboard usability and data exploration.
Identified high-performing and low-performing product categories to support business decision-making.
Demonstrated data storytelling and visualization skills through a clean and user-friendly dashboard design.
